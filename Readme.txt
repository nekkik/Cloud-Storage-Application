Java(TM) SE Runtime Environment (build 14.0.1+7)

To run the program:
Setup:
We need to put the right folder paths for both client and the server programs.

Run:
1. First we need to run the server.java program.
2. Next run the client program multiple times for multiple clients.

Executing the program:
Click on the upload button to upload the file
The textarea would display the file that is uploaded

Click open to open any text file in the editor. We can edit and save as well.
If the sync on toggle button is kept on it would sync to the server.

Click on the sync button to synchronize the file. 

Select any file on the file monitor to get the details of it on the right text area.

Clients can upload any number of files either simultaneously or one after the other.
